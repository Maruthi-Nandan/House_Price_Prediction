# House Price Prediction

## Project Overview
This project aims to predict house prices based on various features such as location, size, and amenities. We utilize machine learning algorithms to build a predictive model that can estimate house prices accurately.

## Features
- Data Collection: Gather data from various sources including online listings and databases.
- Data Preprocessing: Clean and prepare the data for modeling.
- Model Building: Implement machine learning models to predict house prices.
- Evaluation: Use metrics to evaluate the performance of the model.
- Deployment: Create a web application to showcase the model predictions.

## Installation
To run this project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/Maruthi-Nandan/House_Price_Prediction.git
   ```
2. Navigate to the project directory:
   ```bash
   cd House_Price_Prediction
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
1. Run the application using:
   ```bash
   python app.py
   ```
2. Visit `http://localhost:5000` in your browser to see the application.
